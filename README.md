# Text-to-Speech Web Portal

A **Text-to-Speech (TTS)** web portal that allows users to input text, customize the rate and volume of speech, and download the generated audio file. Built with **FastAPI**, **Python**, and **gTTS (Google Text-to-Speech)**.

---

## Features

- **Convert Text to Speech**: Users can input any text, and the application will generate speech from that text.
- **Customizable Speech Rate and Volume**: Adjust the speech rate and volume to your preference.
- **Download Audio**: Users can download the generated speech as an MP3 file.
- **Responsive Web Interface**: The interface is designed to be simple, clean, and responsive for all device sizes.

---

## Technologies Used

- **FastAPI**: The web framework used for the backend, allowing the development of fast, scalable APIs.
- **gTTS**: Google Text-to-Speech (gTTS) is used to convert text into speech and save the audio.
- **Python**: The programming language for the backend logic.
- **HTML/CSS**: Used for building and styling the web interface.
- **JavaScript**: Handles the dynamic interaction on the frontend, including form submission and updating the UI.
- **Uvicorn**: ASGI server used to serve the FastAPI application.
